# Start
`@toriime/types` zawiera wewnętrzne typy używane w projekcie Toriime.

## Instalacja

::: code-group

```bash [npm]
$ npm install @toriime/types
```

```bash [yarn]
$ yarn install @toriime/types
```

```bash [pnpm]
$ pnpm install @toriime/types
```
:::

### Użycie
```ts
// Przykładowy typ
import { Anime } from '@toriime/types';
```
